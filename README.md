# InventoryManager.Mvc

Projeto em ASP.NET Core MVC com funcionalidades de CRUD (Criar, Listar, Editar e Excluir).

## Tecnologias
- ASP.NET Core MVC
- C#
- Razor
- Git e GitHub

## Funcionalidades
- Listar itens
- Criar item
- Editar item
- Excluir item

## Como executar
1. Abrir o projeto no Visual Studio
2. Executar com F5
3. Acessar pelo navegador
